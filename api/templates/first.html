JNTUK previous year papers Crawler



JNTUK PREVIOUS YEAR PAPERS

The QUESTION PAPERS CRAWLER was created by combining the REQUESTS (a Python programme) that gathers data from the internet with the BEAUTIFUL SOUP (a Python tool) that parses the material that was gathered. The web content (HTML, CSS) is then provided using FLASK (a backend framework).

"FETCH UP" is a user-friendly website for retrieving previous year JNTUK B TECH exam papers. Anyone who needs the content can visit the website. When a user wants a reference to model questions, he may enter the website, pick the appropriate department, year, and semester from the drop down menu, and download the paper according to his desired regulation, such as R16, R19, R20, and so on.
